This framework was created to use the Strava API inside your C# application. It was very important to me, that the framework is as easy to use as possible. All you need to do is to register your app with Strava so you get a client id and a client secret. Once you've registered your application, you can start using the framework to develop your Strava Application based on C#. After getting an access token from Strava you can use the StravaClient class to use Strava and get some data. You can find more information about the authentication process in the "Authentication" section. Both the framework and the website are work in progress!
